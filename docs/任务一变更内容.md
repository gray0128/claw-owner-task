# 任务一变更内容：提取公共工具函数

## 1. 变更模块与新增内容
- **`src/worker/utils.ts`**: 新增并导出了以下通用工具函数，将其作为项目级公共模块：
  - `apiResponse`: 统一的 API 响应格式构造函数，替代了各模块中重复写的 `response()`。
  - `toSqliteUtc`: 将前端及系统中用到的时间 Date 对象转化为 SQLite 兼容的 `YYYY-MM-DD HH:mm:ss` UTC 时间字符串。
  - `extractAIContent`: 强化的内容提取器，负责从 Cloudflare AI 可能返回的多种响应结构中稳定提取出文本内容。

## 2. 代码重构与重复消除
- **Handler 与 Service 替换**：清理了以下 7 个文件中重复定义的 `response()` 函数，统一转为从 `utils.ts` 引入别名 `apiResponse as response`，保持现有调用方法无需大量修改：
  - `categories.ts`
  - `tags.ts`
  - `logs.ts`
  - `remind.ts`
  - `tasks.ts`
  - `summary.ts`
  - `ai.ts`
- **时间转换函数去重**：在 `tasks.ts` 中移除了其私有的 `toSqliteUtc` 定义。在 `services/recurrence.ts` 文件中删除了等效的 `formatSqliteDate` 内部函数，统一替换为了 `utils.ts` 导出的 `toSqliteUtc`。
- **AI 解析结构优化**：在 `ai.ts` 与 `summary.ts` 中去除了长片段的判断组装返回内容的 if-else 分支结构，直接接入调用了 `extractAIContent(aiResponse)` 进行简化。
